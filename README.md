# Universal Translators' Tool - программа для анализа, редактирования, импорт и экспорта данных из ресурсов игр, преимущественно из старых игр.

Автор программы: **Илья "Steel Rat" Павловец**


Что программа умеет:
* обрабатывать ресурсы, тексты, шрифты, графику игр.
* расширять свою функциональность с помощью модулей на Lua.

Пожелания и предложения можно писать сюда: http://www.old-games.ru/forum/showthread.php?t=55204

Для сборки необходимы Lua 5.2, OOLUA 2.0, wxWidgets 3.1
